Visualisation of Real-Time Web Data

Mario Weber

Start

track_changes

Idea

Collect, store and visualize data in real-time. The website of Basels public transportation company is periodically checked for delays on a single station. The generated data is stored, analyzed and immediately visualized.

visibility

Visualisations

Scripts periodically transform the received data into JSON objects, which are used to generate different visualisations.

settings

Tools

Several tools are used to support the realisation of this project. The backend is mostly operated by Node.js and Firebase, frontend visualisations in d3.
